Command line
============

Usage: ``roto <COMMAND>``

Commands
--------

``doc``
    Generate documentation for the runtime
``check``
    Type check a script
``test``
    Test a script
``run``
    Run a script
``print``
    Print a Roto file with syntax highlighting
``help``
    Print this message or the help of the given subcommand(s)

Arguments
---------

``-h, --help``
    Print help
``-V, --version``
    Print version
